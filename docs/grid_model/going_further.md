# Going further

To go further with the PowSyBl grid model, have a look at the [PowSyBl tutorials](https://powsybl.readthedocs.io/projects/powsybl-tutorials/), which show step by step how to create and manipulate a network with the Java API, and at the [PyPowSyBl documentation](https://powsybl.readthedocs.io/projects/pypowsybl/) to work with the grid model from Python.
